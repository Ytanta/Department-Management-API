# Department Management API

RESTful API на Go для управления древовидной структурой подразделений и сотрудниками компании.  
Проект реализован с учетом высокой вложенности, защиты от циклов и гибких стратегий удаления.

---

## 🚀 Основные возможности

### Иерархия отделов
Построение дерева любой глубины с использованием `parent_id`.

### Динамическое дерево
Получение структуры подразделений через `GET` с управляемой глубиной (`depth`).

### Умное удаление
Поддержка режимов:

- `cascade` — удаление всего поддерева
- `reassign` — перенос сотрудников

### Валидация
Защита от циклов:
нельзя сделать отдел дочерним по отношению к самому себе или своим потомкам.

### Целостность данных
Все критические операции обернуты в транзакции базы данных.

## 🛠 Технологический стек

- **Language:** Go 1.22+
    - используется стандартный pattern matching в `net/http`
- **Database:** PostgreSQL
- **ORM:** GORM
- **Migrations:** Goose
- **Containerization:** Docker & Docker Compose

## 📦 Запуск проекта

### 1. Требования

Установленные:

- Docker
- Docker Compose

### 2. Быстрый старт

Выполните команду в корневой папке проекта:

```bash
docker-compose up -d --build
```

### Эта команда:

- Поднимет контейнер с PostgreSQL
- Применит все миграции через Goose
- Скомпилирует и запустит Go-приложение

Сервис будет доступен по адресу:

```text
http://localhost:8080
```

## 📡 API Эндпоинты

### Подразделения (Departments)

| Метод | Путь | Описание |
|---|---|---|
| POST | `/departments` | Создание нового отдела |
| GET | `/departments/{id}` | Получение данных отдела, сотрудников и поддерева |
| PATCH | `/departments/{id}` | Обновление имени или перемещение в другой отдел |
| DELETE | `/departments/{id}` | Удаление отдела (`mode=cascade/reassign`) |

### Сотрудники (Employees)

| Метод | Путь | Описание |
|---|---|---|
| POST | `/departments/{id}/employees` | Добавление сотрудника в отдел |
| GET | `/employees/{id}` | Получение данных сотрудника |
| PATCH | `/employees/{id}` | Обновление данных сотрудника |
| DELETE | `/employees/{id}` | Удаление сотрудника |

## 🧪 Примеры запросов (PowerShell)

### Получение дерева глубиной 3 уровня

```powershell
irm "http://localhost:8080/departments/1?depth=3&include_employees=true" | ConvertTo-Json -Depth 10
```

### Каскадное удаление отдела

```powershell
irm "http://localhost:8080/departments/4?mode=cascade" -Method Delete
```
